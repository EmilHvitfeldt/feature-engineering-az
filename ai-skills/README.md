# AI Skills for Feature Engineering A-Z

This folder contains AI-powered skills that readers can use to interact with the book content in new ways. These skills are designed to be used with Claude Code or similar AI assistants.

## Available Skills

- `ask-feaz.md` - Ask questions about feature engineering methods

## Usage

These skills are not part of the rendered book. They are meant to be accessed directly from this GitHub repository and used with AI coding assistants.

## Adding New Skills

To add a new skill, create a new markdown file in this folder following the template structure in `empty-skill.md`.
